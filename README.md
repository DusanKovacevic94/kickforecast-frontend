# KickForecast Frontend

This is the frontend application for KickForecast, built with [Next.js](https://nextjs.org). It provides a responsive, dark-themed user interface for viewing forecasts and managing content via the CMS.

## Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

## Setup

1.  **Navigate to the frontend directory:**
    ```bash
    cd kickforecast-frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

## Running the Application

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Utility functions and API clients.
- `public`: Static assets.
